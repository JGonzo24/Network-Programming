\doxysubsection{Detailed Description}
\doxylink{classinfoSeqNo}{info\+Seq\+No} -\/ maintains a log of all sequence numbers passed in

No changes will be made to the buffers passed to this class, just count the sequence numbers (which are assumed to be in the first 4-\/bytes of each packet in network order.)

Upon destruction, this class will call it\textquotesingle{}s own report function in order.
