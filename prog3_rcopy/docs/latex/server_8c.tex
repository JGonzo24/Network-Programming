Simple UDP server that listens for messages and echoes back byte counts.

This server sets up a UDP socket (IPv6-\/capable), receives messages until it sees a single dot (\textquotesingle{}.\textquotesingle{}) as the first character, and for each message prints the client address + payload, then echoes back the number of bytes received.
