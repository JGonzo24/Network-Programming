Detailed Description

Simple UDP server that listens for messages and echoes back byte counts.

Author
Joshua Gonzalez
Version
0.1
Date
2025-05-10

This server sets up a UDP socket (IPv6-capable), receives messages until it sees a single dot ('.') as the first character, and for each message prints the client address + payload, then echoes back the number of bytes received.
