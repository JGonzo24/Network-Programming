53 /*
54 * In your main()
55 *
56 * To use the sendErr(...) function you must call sendErr_init(...) first.
57 * sendErr_init(...) initializes fields that are used by sendErr(...). You
58 * should only call sendErr_init(...) once in each program.
59 *
60 * sendErr_init(...)
61 * error_rate - between 0 and less than 1 (0 means no errors)
62 * drop_flag - should packets be dropped (DROP_ON or DROP_OFF)
63 * flip_flag - should bits be flipped (FLIP_ON or FLIP_OFF)
64 * debug_flag - print out debug info (DEBUG_ON or DEBUG_OFF)
65 * random_flag - if set to RSEED_ON, then seed is randon, if RSEED_OFF then seed is not random
66 * (see man srand48 - RSEED_OFF makes your program runs more repeatable)
67 *
68 * If you don't know what to set random_flag to, use RSEED_OFF for initial
69 * debugging. Once your program works, try RSEED_ON. This will make the
70 * drop/flip pattern random between program runs.
71 *
72 * ex:
73 * sendErr_init(.1, DROP_ON, FLIP_OFF, DEBUG_ON, RSEED_ON);
74 */
